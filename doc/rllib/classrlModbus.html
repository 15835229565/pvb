Detailed Description

This class implements the modbus protocol.
You can use serial interfaces or TCP/IP.
Modbus RTU and ASCII are available.

          return byte_count;
        default:
          return MODBUS_ERROR;
      }
    }
  }
  else return MODBUS_ERROR;
  return MODBUS_SUCCESS;
